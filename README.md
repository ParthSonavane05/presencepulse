# 📡 Presence Pulse

An intelligent attendance and presence tracking system designed for modern environments.

---

## 🌍 Problem Statement

Traditional attendance systems are inefficient, prone to manipulation (proxy attendance), and lack real-time tracking. Institutions and organizations struggle with accuracy, transparency, and automation.

---

## 💡 Solution

Presence Pulse provides a smart, automated attendance system that leverages real-time tracking and modern technologies to ensure accurate and efficient presence monitoring.

---

## 🚀 Key Features

- 📍 Real-time presence tracking  
- 🧠 Smart attendance detection  
- 📊 Dashboard for monitoring attendance  
- 🔐 Secure and reliable data handling  
- ⚡ Fast and user-friendly interface  

---

## 🛠 Tech Stack

- **Frontend:** Flutter  
- **Backend:** Flask  
- **Database:** Firebase 
- **Version Control:** GitHub  

---

## 🏗 System Architecture

- Mobile App (Flutter) → User interaction  
- Backend Server → Logic & processing  
- Database → Store attendance data

🚀 How to Clone and Run Urja Setu Project

✅ STEP 1: Install Required Tools

Make sure these are installed:
	•	Flutter SDK
	•	Git

Check:
```bash
flutter doctor
```

📥 STEP 2: Clone the Project

Open Terminal and run:
```bash
git clone https://github.com/ParthSonavane05/presencepulse.git
```

📁 STEP 3: Go Inside Project Folder
```bash
cd presencepulse
```

📦 STEP 4: Install Dependencies
```bash
flutter pub get
```

▶️ STEP 5: Run the App
```bash
flutter run
```
